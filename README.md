# GooPet
概述：一款神奇又有趣的桌面宠物App
## 开发目标：
- 具有多种动效、表情
- 可自定义名字、技能、属性等
- 具有闹钟提醒、天气等额外小功能
- 随机启动
- 不进入微信，直接提醒、展示微信消息

## 开发进度：
- 实现视觉差欢迎界面
- 显示宠物，拥有简单交互功能
- 尚在努力开发中，敬请期待~
